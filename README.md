# Data Unification & Clustering
My school homework from **#DescomplicaDigital** on Data Unification and Clustering. Data unification is the process of combining fragmented information from multiple, disparate sources into a single, consistent, and trustworthy view (a _single source of truth_). Clustering is an unsupervised machine learning technique used to group similar data points into clusters based on shared characteristics or proximity measures.
